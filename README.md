## Run Locally 🚀

Clone the project

```bash
  git clone https://github.com/jmr85/07-Primera-entrega-proyecto-final.git
```

Go to the project directory

```bash
  cd 07-Primera-entrega-proyecto-final
```

Install dependencies

```bash
  npm install
```

React app Start 

```bash
  npm start
```
